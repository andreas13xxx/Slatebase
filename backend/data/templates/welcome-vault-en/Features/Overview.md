# Features — Overview

Slatebase offers a wide range of features to help you organize, connect, and rediscover your knowledge. Here you'll find all feature guides at a glance.

## Linking & Embedding

These features form the foundation of a connected knowledge base. With wikilinks you connect ideas, with embeds you inline content, and the knowledge graph makes the structure visible.

| Guide | Description |
|-------|-------------|
| [[Features/Wikilinks\|Wikilinks]] | Link notes together — aliases, heading links, auto-resolve |
| [[Features/Embeds\|Embeds]] | Embed images, PDFs, and notes inline |
| [[Features/Knowledge Graph\|Knowledge Graph]] | Visually explore your knowledge network |

## Organization & Metadata

Bring order without rigid hierarchies. Tags enable cross-cutting categories, properties store structured data, and search finds everything instantly.

| Guide | Description |
|-------|-------------|
| [[Features/Tags and Properties\|Tags and Properties]] | Categorize content with tags and YAML frontmatter |
| [[Features/Context Panel\|Context Panel]] | Outline, links, tags, and properties at a glance |
| [[Features/Search and Replace\|Search and Replace]] | Full-text search, regex, and batch replace |

## Display & Diagrams

Markdown doesn't have to look plain. With callouts you highlight information, with Mermaid you create professional diagrams — all directly in text.

| Guide | Description |

## How the Guides are Structured

Each feature guide follows the same structure:

1. **Brief description** — What does the feature do in 1–2 sentences?
2. **Step by step** — How do you use it?
3. **Practical example** — A concrete scenario to follow along
4. **Tips** — Best practices and shortcuts
5. **Exercise** — Try it directly in the vault
6. **Related features** — Further guides
