# Keyboard Shortcuts

The most important key combinations for efficient work.

## Navigation

| Action | Shortcut |
|--------|----------|
| Command Palette | `Ctrl+P` |
| Search | `Ctrl+Shift+F` |
| Settings | `Ctrl+,` |

## Editor

| Action | Shortcut |
|--------|----------|
| Save | `Ctrl+S` |
| Bold | `Ctrl+B` |
| Italic | `Ctrl+I` |
| Undo | `Ctrl+Z` |
| Redo | `Ctrl+Shift+Z` |

## View

| Action | Shortcut |
|--------|----------|
| Edit Mode | `Ctrl+E` |
| Graph View | `Ctrl+G` |

> [!info] Customizable
> Keyboard shortcuts can be individually configured in the settings.
