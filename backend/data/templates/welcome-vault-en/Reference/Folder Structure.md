# Folder Structure

A good folder structure helps you find notes quickly.

## Recommended Structure

```
My Vault/
├── Projects/        — Active projects
├── Reference/       — Reference material
├── Daily/           — Daily notes
├── Archive/         — Completed items
└── Appendix/        — Images and files
```

## Best Practices

> [!tip] Less is more
> Start with a few folders and add more as needed.

- Use [[Basics/Wikilinks]] instead of deep nesting
- Use [[Basics/Tags and Metadata|Tags]] for flexible categorization
- Keep folder depth at 2-3 levels maximum
