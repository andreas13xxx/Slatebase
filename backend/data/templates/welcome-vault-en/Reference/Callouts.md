# Callouts

Callouts are colored highlight boxes that emphasize important information.

## Syntax

```markdown
> [!type] Title
> Content of the callout
```

## Available Types

> [!tip] Tip
> Use tips for helpful hints and recommendations.

> [!warning] Warning
> Warnings point out potential problems.

> [!info] Information
> Info blocks provide additional context.

## Collapsible Callouts

Add `+` or `-` after the type to make callouts collapsible:

```markdown
> [!tip]+ Expandable (open by default)
> This content can be collapsed.
```
