# Embeds

With embeds you can display images and other notes directly within a note.

## Image Embed

Embed an image:

```markdown
![[Images/example.png]]
```

Preview:

![[Images/example.png]]

## Note Embed

Embed another note:

```markdown
![[Basics/Wikilinks]]
```

## Embed with Size

```markdown
![[Images/example.png|200]]
```

This shows the image at 200px width.
