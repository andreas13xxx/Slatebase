# Markdown Syntax

Markdown is a simple markup language for text formatting.

## Headings

Use `#` for headings:
- `# Heading 1`
- `## Heading 2`
- `### Heading 3`

## Text Formatting

- **Bold** — `**text**`
- *Italic* — `*text*`
- ~~Strikethrough~~ — `~~text~~`
- `Code` — `` `code` ``

## Lists

- Bullet lists with `-` or `*`
- Numbered lists with `1.`, `2.`, etc.

## Links

- External link: `[Title](https://example.com)`
- Internal link: `[[Note title]]` (see [[Basics/Wikilinks]])
