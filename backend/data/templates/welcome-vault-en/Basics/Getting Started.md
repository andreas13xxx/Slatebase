# Getting Started

Welcome to Slatebase! This page gives you an overview of the basics and suggests a sensible learning order.

*The Slatebase interface at a glance*

## Recommended Order

Work through the following guides one by one. Each builds on the previous:

### 1. [[Basics/Markdown Syntax|Markdown Syntax]]

Learn the formatting language you use to write in Slatebase. Headings, lists, tables, and code blocks — everything you need for well-structured notes.

### 2. [[Basics/Editor and Viewer|Editor and Viewer]]

Understand the two modes of Slatebase: Edit mode for writing and View mode for reading. You'll also learn about the toolbar, auto-save, and undo/redo.

### 3. [[Basics/Navigation and Tabs|Navigation and Tabs]]

Find out how to keep multiple files open at once, switch between tabs, and organize your workspace.

### 4. [[Basics/File Explorer|File Explorer]]

Learn how to create, rename, move, and organize files and folders. The explorer is your starting point for vault structure.

## What is a Vault?

A vault is your personal storage space for notes — comparable to a notebook. Everything you create in a vault stays organized there and is only visible to you (unless you share it).

Each vault has:
- A **folder structure** for your files
- A **trash** for deleted items
- **Settings** per vault (templates directory, daily notes)

You can create multiple vaults for different purposes (work, personal, projects).
