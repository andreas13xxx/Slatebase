# Tags and Metadata

Tags and YAML frontmatter help you categorize and filter your notes.

## Tags

Use `#` followed by the tag name: #example #organization

Tags can also be nested: #project/active

## YAML Frontmatter

At the beginning of a note you can specify metadata in YAML format:

```yaml
---
tags:
  - project
  - important
status: active
---
```

This metadata appears in the Properties panel and can be used for filtering.
