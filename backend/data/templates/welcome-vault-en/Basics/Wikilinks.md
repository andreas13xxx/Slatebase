# Wikilinks

Wikilinks connect your notes into a knowledge network.

## Syntax

- `[[Note name]]` — Link to another note
- `[[Folder/Note name]]` — Link with path
- `[[Note name|Display name]]` — Link with alternative text
- `[[Note name#Heading]]` — Link to a heading

## Examples

- Back to the start page: [[Start here]]
- Link with path: [[Reference/Callouts]]
- Link with display name: [[Projects/Example Project|My Project]]

## Benefits

Wikilinks automatically create connections between notes. This builds a knowledge graph that you can explore in the Graph View.
