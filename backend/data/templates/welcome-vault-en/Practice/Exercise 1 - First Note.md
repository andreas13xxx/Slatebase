# Exercise 1 — Create Your First Note

**Difficulty:** :star: Easy
**Duration:** ~5 minutes

---

## Goal

You'll create your first file, format it with Markdown, and view the result in the preview.

## Prerequisites

- You're logged into Slatebase
- You have a vault open (e.g., this Welcome vault)

---

## Steps

> [!todo] Step 1: Create a new folder
> 1. Right-click in the file explorer on an empty area
> 2. Select **New Folder**
> 3. Name the folder `Sandbox`
> 4. Confirm with Enter

> [!todo] Step 2: Create a new file
> 1. Right-click on the `Sandbox` folder
> 2. Select **New File**
> 3. Name the file `My Note`
> 4. The file opens automatically in the editor

> [!todo] Step 3: Write Markdown
> Write the following content in the file:
>
> ```markdown
> # My First Note
>
> This is my first attempt with **Slatebase**!
>
> ## What I Learned Today
>
> - Creating files
> - Formatting text with *Markdown*
> - Writing bold and italic
>
> ## Next Steps
>
> 1. Create links to other notes
> 2. Try out tags
> 3. Explore the graph
> ```

> [!todo] Step 4: View the preview
> 1. Switch from **Edit mode** to **View mode** (button in the toolbar or `Ctrl+E`)
> 2. You'll now see the formatted view:
>    - Headings are displayed larger
>    - **Bold** and *italic* are rendered correctly
>    - Lists have bullets or numbers

> [!todo] Step 5: Confirm save
> The file is saved automatically (auto-save). You can recognize the save status from the tab bar:
> - No dot = saved
> - Dot on tab name = unsaved changes

---

## Success Criteria

- [ ] The folder `Sandbox` exists in the file explorer
- [ ] The file `Sandbox/My Note.md` is present
- [ ] The file contains a heading, bold text, and a list
- [ ] In View mode everything is displayed correctly formatted

---

## What You Learned

- Creating folders and files via the context menu
- Basic Markdown syntax: headings (`#`), bold (`**`), italic (`*`), lists (`-`, `1.`)
- Switching between Edit and View mode
- Recognizing auto-save
