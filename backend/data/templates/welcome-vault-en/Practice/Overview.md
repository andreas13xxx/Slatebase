## Tips

> [!tip] Sandbox Folder
> All exercises create new files in a `Sandbox/` or `Projects/` folder. Your existing notes are **not modified**.

> [!info] Order
> The exercises build on each other. Start with Exercise 1, even if you already have experience — later exercises reference files from earlier ones.

---

## How to Use the Exercises

1. Read the **Goal** section — what will you be able to do at the end?
2. Check the **Prerequisites** — have you done the previous exercise?
3. Follow the **Steps** (numbered, in `> [!todo]` boxes)
4. Check the **Success Criteria** at the end — all done?

> [!warning] Note
> The exercises create new files and folders. Existing content is **not modified or deleted**. You can safely complete every exercise.
