# Exercise 5 — Create a Canvas

**Difficulty:** :star::star::star: Advanced
**Duration:** ~15 minutes

---

## Goal

You'll create a visual canvas with text nodes, file nodes, groups, and connecting edges — building a brainstorming board.

## Prerequisites

- Exercises 1–3 completed (you have project files to reference)
- You know the basics of the [[Features/Canvas|Canvas]] feature

---

## Steps

> [!todo] Step 1: Create a canvas file
> 1. Right-click in the `Sandbox` folder → **New Canvas**
> 2. Name it `Brainstorm`
> 3. The canvas editor opens with an empty board

> [!todo] Step 2: Add text nodes
> 1. Double-click on the empty canvas — a text node appears
> 2. Type: "Main Topic: Learn Slatebase"
> 3. Create 3 more text nodes around it:
>    - "Markdown Basics"
>    - "Link Everything"
>    - "Organize with Tags"
> 4. Position them around the central node

> [!todo] Step 3: Connect with edges
> 1. Hover over the central node — anchor points appear
> 2. Click and drag from an anchor to "Markdown Basics"
> 3. Release — an edge is created
> 4. Connect the center to all three surrounding nodes
> 5. Right-click an edge → add a label (e.g., "step 1")

> [!todo] Step 4: Add a file node
> 1. Drag `Sandbox/My Note.md` from the file explorer onto the canvas
> 2. A file node appears showing the note's content
> 3. Connect it to "Markdown Basics" with an edge
> 4. Resize the file node by dragging its corners

> [!todo] Step 5: Create a group
> 1. Use the toolbar to add a Group node
> 2. Make it large enough to contain 2–3 nodes
> 3. Drag "Markdown Basics" and the file node into the group
> 4. Label the group "Fundamentals"
> 5. Try coloring the group (right-click → Color)

> [!todo] Step 6: Explore canvas tools
> 1. Use the scroll wheel to zoom in and out
> 2. Click and drag on empty space to pan
> 3. Click "Fit to view" in the toolbar
> 4. Toggle the minimap and navigate by clicking on it
> 5. Try the source view to see the raw JSON

---

## Success Criteria

- [ ] A canvas file `Sandbox/Brainstorm.canvas` exists
- [ ] At least 4 text nodes are placed on the canvas
- [ ] Edges connect the nodes
- [ ] At least one file node references an existing file
- [ ] A group contains multiple nodes
- [ ] You've used zoom, pan, and minimap

---

## What You Learned

- Creating canvas files
- Adding text nodes, file nodes, and groups
- Connecting nodes with edges (including labels)
- Navigating the canvas (zoom, pan, minimap)
- Using the canvas toolbar (fit, source view)
