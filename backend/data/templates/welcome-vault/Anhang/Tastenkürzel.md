# Tastenkürzel

Die wichtigsten Tastenkombinationen für schnelles Arbeiten.

## Navigation

| Aktion | Kürzel |
|--------|--------|
| Befehlspalette | `Strg+P` |
| Suche | `Strg+Shift+F` |
| Einstellungen | `Strg+,` |

## Editor

| Aktion | Kürzel |
|--------|--------|
| Speichern | `Strg+S` |
| Fett | `Strg+B` |
| Kursiv | `Strg+I` |
| Rückgängig | `Strg+Z` |
| Wiederherstellen | `Strg+Shift+Z` |

## Ansicht

| Aktion | Kürzel |
|--------|--------|
| Bearbeitungsmodus | `Strg+E` |
| Graph-Ansicht | `Strg+G` |

> [!info] Anpassbar
> Tastenkürzel können in den Einstellungen individuell konfiguriert werden.
