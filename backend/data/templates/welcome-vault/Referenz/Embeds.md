# Embeds

Mit Embeds kannst du Bilder und andere Notizen direkt in eine Notiz einbetten.

## Bild-Embed

Ein Bild einbetten:

```markdown
![[Anhang/Bilder/beispiel.png]]
```

Vorschau:

![[Anhang/Bilder/beispiel.png]]

## Embed mit Größenangabe

Du kannst die Breite in Pixeln angeben:

```markdown
![[Anhang/Bilder/beispiel.png|100]]
```

Vorschau (100px breit):

![[Anhang/Bilder/beispiel.png|100]]

## Notiz-Embed

Eine andere Notiz direkt einbetten:

```markdown
![[Grundlagen/Wikilinks]]
```

Vorschau:

![[Grundlagen/Wikilinks]]

## Embed eines bestimmten Abschnitts

Du kannst auch nur einen Abschnitt einer Notiz einbetten:

```markdown
![[Grundlagen/Markdown Syntax#Überschriften]]
```

> [!tip] Tipp
> Embeds sind ideal, um wiederkehrende Inhalte zentral zu pflegen und an mehreren Stellen einzubinden.
