# Embeds

Mit Embeds kannst du Bilder und andere Notizen direkt in eine Notiz einbetten.

## Bild-Embed

Ein Bild einbetten:

```markdown
![[Bilder/beispiel.png]]
```

Vorschau:

![[Bilder/beispiel.png]]

## Notiz-Embed

Eine andere Notiz einbetten:

```markdown
![[Grundlagen/Wikilinks]]
```

## Embed mit Größenangabe

```markdown
![[Bilder/beispiel.png|200]]
```

Dies zeigt das Bild mit 200px Breite.
