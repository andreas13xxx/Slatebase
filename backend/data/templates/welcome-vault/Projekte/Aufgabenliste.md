# Aufgabenliste

Nutze Checkboxen, um Aufgaben direkt in Markdown zu verwalten.

## Offene Aufgaben

- [ ] Slatebase-Grundlagen lernen
- [ ] Eigene Notizen erstellen
- [ ] Ordnerstruktur einrichten
- [ ] Tags-System definieren

## Erledigte Aufgaben

- [x] Welcome-Vault erkunden
- [x] [[Start hier]] lesen

## Syntax

Eine Aufgabe erstellst du mit `- [ ]` (offen) oder `- [x]` (erledigt).

#aufgaben #organisation
