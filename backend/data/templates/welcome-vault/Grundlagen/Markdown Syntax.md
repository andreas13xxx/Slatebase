# Markdown Syntax

Markdown ist eine einfache Auszeichnungssprache für Textformatierung.

## Überschriften

Verwende `#` für Überschriften:
- `# Überschrift 1`
- `## Überschrift 2`
- `### Überschrift 3`

## Textformatierung

- **Fett** — `**Text**`
- *Kursiv* — `*Text*`
- ~~Durchgestrichen~~ — `~~Text~~`
- `Code` — `` `Code` ``

## Listen

- Aufzählung mit `-` oder `*`
- Nummerierung mit `1.`, `2.`, etc.

## Links

- Externer Link: `[Titel](https://beispiel.de)`
- Interner Link: `[[Notiztitel]]` (siehe [[Grundlagen/Wikilinks]])
