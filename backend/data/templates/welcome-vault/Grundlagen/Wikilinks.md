# Wikilinks

Wikilinks verbinden deine Notizen zu einem Wissensnetz.

## Syntax

- `[[Notizname]]` — Link zu einer anderen Notiz
- `[[Ordner/Notizname]]` — Link mit Pfad
- `[[Notizname|Anzeigename]]` — Link mit alternativem Text
- `[[Notizname#Überschrift]]` — Link zu einer Überschrift

## Beispiele

- Zurück zur Startseite: [[Start hier]]
- Link mit Pfad: [[Referenz/Callouts]]
- Link mit Anzeigename: [[Projekte/Beispielprojekt|Mein Projekt]]

## Vorteile

Wikilinks erzeugen automatisch Verbindungen zwischen Notizen. So entsteht ein Wissensgraph, den du im Graph-View erkunden kannst.
