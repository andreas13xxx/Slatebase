# Slatebase — Product Overview

Slatebase is a self-hosted Knowledge-Context-Server for Markdown vaults. It allows users to manage, browse, and view Markdown-based knowledge bases through a web interface. The system is designed to be compatible with Obsidian vaults.

## Current State (MVP)

The MVP implements:
- Multi-vault management (create, delete, list)
- File explorer with directory tree navigation
- File viewer for Markdown content (read-only)
- File/folder import into vaults
- Content deletion within vaults

## Planned Features

- Obsidian-compatible Markdown rendering (wikilinks, embeds, frontmatter)
- Tabbed document viewer
- Inline Markdown editor with live preview
- Knowledge graph visualization
- AI Context Server with MCP (Model Context Protocol) integration
- Vault synchronization (LiveSync/CouchDB compatible)
- User authentication and authorization
- Obsidian Community Plugin compatibility layer
- Internationalization (i18n) — German and English
- Accessibility (a11y) — WCAG 2.1 AA compliance

## Language

The product UI currently uses German labels (e.g., "Laden…", "Fehler"). Requirements and documentation are written in German. Code comments and identifiers are in English.
